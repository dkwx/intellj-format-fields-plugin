# intellj-format-fields-plugin
 这个插件可以将Java文件的全局字段排序，按代码的长度由短到长排序。     
 This plug-in can sort the global fields of Java files, from short to long, according to the length of the code.    
 For example:     
 int i;     
 @Resource     
 Service service;     
 boolean bool;      
 
 After handle:     
 int i;     
 boolean bool;     
 @Resource     
 Service service;
